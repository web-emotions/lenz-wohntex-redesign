### `/` — Startseite

**Zweck:** Vertrauen aufbauen, regional positionieren, zu Leistungen führen  
**SEO-Fokus:** „Raumausstatter Schwäbisch Hall", „Raumausstatter Michelfeld"

**Sektionen:**
1. **Hero** – Headline + Subline + CTA-Button „Leistungen entdecken" + CTA „Jetzt anrufen"
2. **Intro-Text** – Kurze Unternehmensvorstellung: seit 1985, Showroom Michelfeld, persönliche Beratung
3. **Leistungs-Kacheln** (4) – Gardinen · Bodenbeläge · Sonnenschutz · Insektenschutz (je mit Bild, Kurztext, Link)
4. **Warum Lenz** – 3–4 Argumente: Vor-Ort-Beratung, Aufmaß & Montage, Showroom, seit 1985
5. **Referenz-Teaser** – 3 Ausgewählte Referenz-Fotos + Link zu /referenzen/
6. **Kontakt-Leiste** – Telefon, Adresse, Öffnungszeiten prominent

**Meta-Titel:** Raumausstatter Schwäbisch Hall – Gardinen, Bodenbeläge & mehr | Lenz Wohntex  
**Meta-Description:** Ihr Raumausstatter in Michelfeld seit 1985. Gardinen nach Maß, Bodenbeläge, Sonnenschutz und Insektenschutz – persönliche Beratung, Aufmaß und Montage. Jetzt anrufen.

## Technische Anforderungen

| Bereich | Lösung |
|---------|--------|
| Framework | Astro 4.x (Static Site Generation) |
| Styling | CSS Custom Properties + vanilla CSS (kein Tailwind, kein Framework) |
| Fonts | Google Fonts oder selbst gehostet (DSGVO-konform) |
| Bilder | Astro `<Image>` mit WebP-Optimierung + lazy loading |
| Kontaktformular | Netlify Forms oder Formspree (kein Backend nötig) |
| Google Maps | Datenschutzkonformes Einbetten (2-Klick-Lösung oder statisches Bild) |
| Redirects | `_redirects`-Datei (Cloudflare Pages) oder `vercel.json` |
| Sitemap | `@astrojs/sitemap` Integration |
| SEO | `@astrojs/sitemap` + individuelle Meta-Tags je Seite |
| Analytics | Datenschutzfreundliche Lösung (z. B. Plausible oder GA4 mit Cookie-Consent) |
| Deployment | GitHub → Cloudflare Pages oder Vercel |
| Cookie-Consent | Einfaches, DSGVO-konformes Banner (ohne externe Bibliothek) |
