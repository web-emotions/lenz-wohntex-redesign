## 7. Content-Qualität & Textempfehlungen

### Zielwortanzahl je Seite (Minimum)
| Seite | Ziel-Wortanzahl |
|-------|----------------|
| Startseite | 400–600 Wörter |
| Leistungen Übersicht | 300–500 Wörter |
| Je Leistungsseite | 500–800 Wörter |
| Referenzen | 150–300 Wörter (Bilder dominieren) |
| Kontakt | 200–300 Wörter |
| Partner | 250–400 Wörter |

### Ton & Stil
- Direkte Ansprache (Sie)
- Keine Floskeln: kein „wir bieten Ihnen hochwertige Qualität zu fairen Preisen"
- Konkrete Produkte, Materialien, Vorteile nennen
- Regionalen Bezug natürlich einbauen
- Vertrauen durch Erfahrung: „seit 1985", „über 35 Jahre", nicht „viele Jahre"
- Handwerk betonen: „wir messen auf, beraten persönlich, verlegen selbst"

### Was vermieden werden soll
- KI-typische Phrasen: „maßgeschneiderte Lösungen", „Ihre Wünsche stehen im Mittelpunkt", „auf höchstem Niveau"
- Übermäßige Superlative: „bestes", „günstigstes", „einzigartig"
- Zu langer Fließtext ohne Struktur

## 8. Technische SEO-Checkliste

| Maßnahme | Umsetzung |
|----------|-----------|
| Meta-Titel (alle Seiten) | Astro BaseLayout, individuelle Props |
| Meta-Description (alle Seiten) | Astro BaseLayout, individuelle Props |
| Kanonische URLs | `<link rel="canonical">` im Head |
| Sitemap XML | `@astrojs/sitemap` automatisch |
| robots.txt | Statische Datei in `/public/` |
| Open Graph Tags | Für Social Sharing |
| Hreflang | Nicht nötig (nur DE) |
| Structured Data | JSON-LD im Head |
| Alt-Texte | Alle Bilder mit beschreibenden Texten |
| Lazy Loading | Astro `<Image>` Component |
| WebP | Astro automatische Konvertierung |
| Seitentitel-Hierarchie | Nur 1x H1 je Seite |
| HTTPS | Cloudflare/Vercel erzwingen |
| Core Web Vitals | Static Site = gute Ausgangslage |
| Mobile-First | CSS-Breakpoints mobile → desktop |
| 301-Redirects | `_redirects`-Datei für alle alten URLs |

## 10. Prioritäten-Reihenfolge für den Launch

**Phase 1 (Launch-Blocker):**
1. Alle Meta-Titel und Meta-Descriptions
2. Impressum bereinigt (nur Lenz Wohntex)
3. Datenschutzerklärung DSGVO-konform
4. 301-Redirects aktiv
5. Alle Alt-Texte gesetzt
6. Sitemap befüllt

**Phase 2 (kurz nach Launch):**
7. LocalBusiness Schema.org aktiv
8. Google Business Profile aktualisieren
9. Google Search Console anmelden + Sitemap einreichen
10. Core Web Vitals prüfen

**Phase 3 (mittelfristig):**
11. Textlänge der Leistungsseiten ausbauen
12. Weitere Referenz-Fotos mit beschreibenden Alt-Texten ergänzen
13. Kundenbewertungen aufbauen (Google)
